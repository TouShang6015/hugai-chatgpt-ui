# hugai-chatgpt-ui前端仓库

## 介绍

HugAi是由Springboot集成openAi SDK开发的一套智能AI知识库，支持GPT对话，AI绘图Midjourney、Stable Diffusion、openai，前后端源码完全开源。

> 声明：基于 Apache-2.0 协议，免费且作为开源学习使用，未经本人同意不可用于商业项目。
